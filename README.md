# shad6cogs
Shadow6six's cogs - they seem to work!

Shadow6six's cogs for [Red-DiscordBot](https://github.com/Cog-Creators/Red-DiscordBot).

## Installation
1. Make sure the Downloader cog is loaded:
[p]load downloader

Add this repository:
[p]repo add shad6cogs https://github.com/Shadcodez/shad6cogs

Install any cog you want:
[p]cog install shad6cogs anisearch
[p]cog install shad6cogs bravesearch
[p]cog install shad6cogs excelevents
[p]cog install shad6cogs musiclinker
[p]cog install shad6cogs pixelart

## Available Cogs - (more to come)
- **anisearch** — Search anime, manga, characters & users on Anilist + MyAnimeList  
- **bravesearch** — Web search + optional Brave AI answers  
- **excelevents** — Bulk create/update/delete Discord scheduled events from an Excel file  
- **musiclinker** — Auto-convert Spotify/YouTube music links + lyrics search  
- **pixelart** — Convert any image into pixel art with live editing
